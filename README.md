# Retromaika — форма заказа v3

## Что изменено в этой версии

1. **Пустой абзац** между «Новый заказ @retromaika» и ФИО — исправлено (`filter(Boolean)` → `filter(x => x !== null)`)
2. **Города** — словарь 100+ официальных английских названий + ГОСТ транслит для остальных
3. **Т-банк API** — платёж создаётся автоматически на нужную сумму, не нужно руками делать ссылку
4. **Сумма в ссылке** — передаёшь `?amount=3500` и клиент видит «Оплатить 3 500 ₽» на кнопке

## Netlify Environment Variables

В **Site settings → Environment variables** добавить:

| Переменная            | Где взять                                      |
|-----------------------|------------------------------------------------|
| `TELEGRAM_BOT_TOKEN`  | BotFather → перевыпустить токен                |
| `TELEGRAM_CHAT_ID`    | ваш chat_id                                    |
| `TBANK_TERMINAL_KEY`  | Личный кабинет Т-банка → Терминалы → TerminalKey |
| `TBANK_SECRET_KEY`    | Личный кабинет Т-банка → Терминалы → Пароль    |

## Как отправлять ссылку клиенту

Договорились на сумму в переписке → отправляешь:

```
https://retromaika.netlify.app/?amount=3500
```

Клиент открывает форму, видит «Оплатить 3 500 ₽», заполняет адрес, нажимает кнопку → автоматически попадает на страницу оплаты Т-банка с нужной суммой.

## Как проверить что всё работает

1. Залить все файлы на Netlify
2. Задать переменные окружения
3. Открыть `?amount=100` (тестовая сумма)
4. Заполнить форму
5. Должно прийти 2 сообщения в Telegram и открыться страница оплаты Т-банка
6. Если ошибка — код ошибки виден под кнопкой, ищите его в Netlify → Functions → Logs
